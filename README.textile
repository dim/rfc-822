h1. RFC822

RFC822 compatible email validation and MX record check.


h2. Features

* Ruby 1.8 & 1.9 compatible
* Email validation
* MX checks (requires 'host' command line application)



h2. Installation

h6. Download from GitHub

bc. wget http://github.com/dim/rfc822/tarball/master

h6. Clone from GitHub

bc. git clone git://github.com/dim/rfc822.git

h6. As a Rails plugin

bc. ruby script/plugin install git://github.com/dim/rfc822.git



h2. Usage Examples

bc. validates_format_of :email, :with => RFC822::EMAIL 

bc. "user@home.domain.com" =~ RFC822::EMAIL ? puts("Email is valid.") : puts("Email is invalid")
